# Playgrounds

This directory contains the DataMaster playgrounds retained for the public MLE-Bench workflow.

| Playground | Purpose |
| --- | --- |
| `ml_master` | Baseline MLE-Bench style machine-learning workflow. |
| `data_master` | Main DataMaster workflow with DataTree, Data Pool, Global Memory, red nodes, and black nodes. |

The playgrounds are registered through EvoMaster's playground registry and can be invoked with `run.py`.
