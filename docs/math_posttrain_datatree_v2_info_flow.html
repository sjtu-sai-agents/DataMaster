Math Posttrain Datatree V2 信息流

本页可视化 global、red node、black node 之间的信息传递，以及 red/black prompt 的主要输入和输出。

Red node
读取 global pool、上游 handoff 和 memory，搜索并注册高质量训练数据。
Black node
读取 global pool 和上游信号，清洗数据、训练、评估，并产出新的 handoff。
Global state
由 global pool、black handoff、node memory 和 data links 组成，跨节点复用。

总体链路

seed 先评估 base model 并写 baseline handoff；red 根据该信号扩充 global_pool_manifest.json；black 消费 global pool，产出训练评估结果和新的 handoff。

Red Prompt 输入与输出

Red prompt 目标是找到和 benchmark row-level distribution 对齐的公共训练数据。 它不做训练改动，只更新全局数据池。

    RedPrompt --> Probe
    Probe --> Manifest
    Manifest --> PoolOut
    RedPrompt --> RedMemory

Black Prompt 输入与输出

Black prompt 目标是把 global pool 里的候选数据转成一次可评估的 post-training 实验。 成功标准是产出有效训练数据并完成 submit。

循环策略

调度关系是 root -> seed -> red -> black。black 完成后，根据 recommended_next_action 继续扩展新的 black 或 red。

    Root --> Seed
    Seed --> Red
    Red --> Black
    Black --> Decision
    Decision -->|expand_black| Black
    Decision -->|expand_red| Red
