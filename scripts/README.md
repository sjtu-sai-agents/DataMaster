# Scripts

This directory keeps small DataMaster maintenance helpers. It intentionally does not include internal experiment launchers or benchmark-ablation shell scripts.

- `auto_config_exp.py`: generate task-specific DataMaster configuration files from templates.
- `build_full_initial_codes.py`: assemble starter-code manifests for MLE-Bench style tasks.
- `check_port_conflicts.py`: inspect local grading-server port usage.
- `prefetch_models.py`: optional helper for preparing local model assets.
